    3.1 Process with unique ID

    • User process associate with a particular user, fg and bg

    • Kernel threads part of the kernel that are running as if they were regular user processes or system daemon, but are not associated with the terminal, they are part of the kernel, but they are still scheduled as if there were regular processes
